# Segundo proyecto Servivcios 
 
 ##Indicaciciones
 1. **Localizar el archivo .env.example**
 2. **Colocar en la variable "AZURE_SPEECH_KEY" la API KEY**
 3. **COPIAR Y RENOMBRAR** el archivo .env.example a .env
    - Esto ayuda a que el archivo .gitignore no suba la API KEY al repositorio

# Instalación por Plataforma/Lenguaje (SDK de Voz)
- python: pip install azure-cognitiveservices-speech
- Node.js / JavaScript: npm install microsoft-cognitiveservices-speech-sdk
